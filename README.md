# CCIP-CCRP-HUD-Demo
a python script using Ursina


conda create -n HUD python=3.10\
pip install ursina\
pip install numpy pillow

w s: throttle and spoiler\
arrow up down left right: pitch and roll\
q: NAV CCIP CCRP mode change\
e: lock/unlock CCRP target\
r: bomb type\
ijkl: CCRP TDC\
space: CCIP release, or get ready to CCRP release with auto LOC guidance
